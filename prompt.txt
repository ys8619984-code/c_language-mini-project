Section 3: Final Code Overview

Project Name:

2D Graphics Editor Using Character Matrix

Language:

C Programming

Purpose:

A console-based graphics editor capable of drawing geometric shapes using ASCII characters.


---

Section 4: Data Structures Used

Enumeration

typedef enum
{
    LINE = 1,
    RECTANGLE,
    CIRCLE,
    TRIANGLE
} ShapeType;

Purpose

Provides identifiers for shape categories.

Benefits:

Better readability

Easier maintenance

Improved organization



---

Section 5: Shape Structures

LineData

typedef struct
{
    int x1,y1,x2,y2;
} LineData;

Stores line endpoints.


---

RectData

typedef struct
{
    int x1,y1,x2,y2;
} RectData;

Stores rectangle corner coordinates.


---

CircleData

typedef struct
{
    int xc,yc,r;
} CircleData;

Stores center and radius.


---

TriData

typedef struct
{
    int x1,y1,x2,y2,x3,y3;
} TriData;

Stores triangle vertices.


---

Section 6: Union Usage

union
{
    LineData line;
    RectData rect;
    CircleData circle;
    TriData tri;
};

Purpose

Stores only one shape type at a time.

Benefit

Memory-efficient implementation.


---

Section 7: Linked List Implementation

ShapeNode

typedef struct ShapeNode
{
    int id;
    ShapeType type;
    ...
    struct ShapeNode* next;
} ShapeNode;

Purpose

Stores graphical objects dynamically.

Benefits

Dynamic storage

Easy insertion

Easy deletion

No fixed limit

Section 8: Canvas System

Canvas declaration:

char canvas[ROWS][COLS];

Purpose

Represents the drawing area.

Symbols:

_  = Empty Space


---

Section 9: Drawing Functions

reset_canvas()

Clears canvas.

Uses:

memset()

Fills canvas with underscores.


---

plot()

Draws a single point.

Checks boundaries before plotting.


---

draw_line()

Uses:

Bresenham Line Algorithm

Purpose:

Draw a straight line between two points.

Advantages:

Fast

Uses integer arithmetic

Accurate



---

draw_rectangle()

Constructed using four lines.


---

draw_circle()

Uses:

Midpoint Circle Algorithm

Advantages:

Efficient

Symmetry-based

No floating-point calculations



---

draw_triangle()

Draws three lines connecting three vertices.


---

Section 10: Rendering System

render_all_shapes()

Purpose:

Reconstructs complete image.

Process:

1. Clear canvas.


2. Traverse linked list.


3. Draw each shape.


4. Display final picture.

Section 16: Project Summary

The final project evolved from a basic syllabus-based graphics editor into a more advanced implementation that supports dynamic object management through linked lists, efficient memory usage through unions, persistent storage through file handling, and shape rendering using Bresenham Line Algorithm and Midpoint Circle Algorithm.

The project fulfills all assignment requirements while demonstrating multiple Advanced C Programming concepts in a practical application.
